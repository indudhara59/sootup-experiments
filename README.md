# SootUp Experiments

A collection of experiments using SootUp for Java program analysis.

## Project Status

Work in progress.

## Goal

The goal of this project is to explore SootUp and understand how it can be used for Java static analysis.

## Technologies

- Java
- Maven
- SootUp

## Planned Features

- Explore class hierarchy analysis
- Generate call graphs
- Analyze method dependencies
- Document SootUp usage examples
- Create small experiments for Java program analysis workflows

## Current Focus

Setting up SootUp examples and documenting initial experiments.
